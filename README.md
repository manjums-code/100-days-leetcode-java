## Why This Challenge?

This challenge is for beginners.

If you've ever wanted to learn Data Structures & Algorithms but felt overwhelmed by the amount of content available, you're not alone.

The goal isn't to solve hundreds of problems overnight.

The goal is to show up every day.

A small amount of consistent effort can create remarkable progress over time.

- Solve LeetCode problems daily
- Learn DSA step-by-step
- Build problem-solving skills
- Stay consistent for 100 days
- Document the entire journey publicly
